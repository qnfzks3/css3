CSS position 관련

개요

지금까지는 margin/padding 또는 인라인/블록 요소의 특성에 의해 부분적으로나 요소의 위치를 지정할 수 있엇음

반면, position이라는 속성을 이용하면 html요소들의 위치를 좌표등으로 결정할 수 있음

즉, top/left/right/bottom 등의 속성으로 요소의 위치를 정할 수 있음

static position

기본적인 요소 배치순서에 따라 위->아래 , 왼쪽->오른쪽으로 배치되며, 부모-자식 관계의 요소는 부모 요소에 따라 배치됨

box#1
box#2
box#3

absolute position

조상요소(body) 또는 부모요소를 기준으로 top/left/right/bottom 만큼 요소를 이동시켜 배치 보통 좌표 (0,0)이 시작 body 좌표다. 부모가 박스가 있다면 부모 박스의 시작 좌표가 바디이다.

단, 부모요소를 기준으로 하위요소를 배치하려면 부모요소의 position은 relative이어야 함

box#4
box#5
box#6
box#7

relative position

대상이 있는 위치를 기준으로 top/left/right/bottom 만큼 요소를 이동시켜 배치

box#8
box#9
box#10
box#11

fixed

뷰포트(웹페이지 가시영역)를 기준으로 top/left/right/bottom만큼 요소를 이동시켜 배치

따라서, 브라우져를 스크롤 하더라도 뷰포트의 특정위치에 요소가 배치됨

sticky position

relative position 과 fixed position을 적절히 조합해서 만든 position

평소에는 relative 방식으로 요소를 처리하다가 해당 요소가 뷰포트를 넘으면 fixed처럼 화면에 고정 시킴

자신의 스테틱 위치에 오면 그 후로 fix처럼 화면에 고정시킴

sticky

z-index

요소를 배치하다보면 설정된 위치나 방식에 따라 서로 겹칠 수 있음

즉, 먼저 만들어진 요소는 나중에 만들어진 요소보다 밑(뒤)에 배치됨

요소가 쌓이는 순서를 지정하고 싶다면 z-index를 지정하면 됨

z-index값이 클수록 앞쪽에 위치함
