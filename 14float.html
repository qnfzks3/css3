css 테두리 영역의 크기를 결정하는 속성

일반적으로 블록 요소의 크기는 바깥여백 + 테두리 + 안쪽여백을 합산한 값으로 결정

따라서, 기본적으로 설정한 width, height 외 바깥여백 + 테두리 + 안쪽여백까지 계산 되므로 실제 크기보다 더 커지게 되는 상황이 발생 - 자칫 레이아웃을 엉망으로 만들 수도 있음

한편, border-box 속성을 이용하면 블록 요소의 크기를 계산할 때 전체 영역을 대상으로 바깥여백+테두리+안쪽여백을 포함시킨

단, 컨텐츠 영역은 실제 설정값 보다 작아질 수도 있음

float

clear

float 속성이 지정된 요소는 문서의 일반적인 흐름상에서 벗어난 상태이기 때문에 뒤를 따르는 후속요소들은 다소 이상한 모습을 보여줌

이러한 문제를 해결하기 위해서 clear 속성을 적용해 특정 요소 이후로는 float의 영향을 받지 않도록 해야 함
